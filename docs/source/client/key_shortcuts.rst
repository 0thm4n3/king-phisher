Keyboard Shortcuts
==================

+--------------------+-------------------------------+
| Key Combination    | Action Description            |
+====================+===============================+
| ``Ctl`` + ``O``    | Open a campaign               |
+--------------------+-------------------------------+
| ``Ctl`` + ``Q``    | Exit the client               |
+--------------------+-------------------------------+
| ``Ctl`` + ``F1``   | Open an RPC terminal          |
+--------------------+-------------------------------+
| ``Ctl`` + ``F2``   | Open the SFTP client          |
+--------------------+-------------------------------+
